# Support

## Getting Help

### Documentation

Full documentation is available at [veritera.ai](https://veritera.ai).

### Community

- **GitHub Issues** -- Bug reports and feature requests for SDKs and adapters
- **GitHub Discussions** -- Questions, ideas, and community support

### Enterprise Support

Enterprise customers receive dedicated support. Contact [enterprise@veritera.ai](mailto:enterprise@veritera.ai).

## Reporting Bugs

When filing a bug report, please include:

1. SDK or adapter name and version
2. Runtime environment (Node.js version, Python version, OS)
3. Steps to reproduce
4. Expected behavior vs. actual behavior
5. Error messages or logs (redact any API keys)

## Feature Requests

Open a GitHub Issue with the `enhancement` label. Describe:

- The problem you are trying to solve
- Your proposed solution
- Any alternatives you have considered

## Security Issues

For security vulnerabilities, see [SECURITY.md](SECURITY.md). Do not open a public issue for security reports.
